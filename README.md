# 🦷 Dental Clinic Website

A modern, fully responsive **Dental Clinic management website** built with **React**, featuring an intuitive user experience for patients and powerful admin tools for managing the clinic efficiently.

---

## ✨ Features

### 👥 User Side
- 🪥 **Services Page** – Explore all dental services with detailed information.  
- 📅 **Booking System** – Patients can easily book appointments online.  
- 🌍 **Language Translation** – Seamless multilingual support for a wider audience.  
- 📞 **Contact Form** – Reach out directly through the website.  
- 📱 **Fully Responsive** – Optimized for desktop, tablet, and mobile devices.

### 🧑‍💼 Admin Side
- 🧾 **Dashboard Overview** – See total users, appointments, and messages at a glance.  
- 🩺 **Doctor Management** – Add, edit, or remove doctors dynamically.  
- 📋 **Appointments Management** – View and manage patient bookings.  
- ✉️ **Contacts Management** – Track and respond to contact form submissions.
